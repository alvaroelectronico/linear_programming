Dado el siguiente problema, $P$, de programación lineal:

\begin{equation}
\begin{split}
\mbox{max. } z = x_1 + 4x_2\\
s.a.:\\
4x_1 + 2x_2 = 80\\
2x_1 + 3x_2 \leq 60\\
x_1,\,x_2\geq 0
\end{split}
\end{equation}

Se pide:

\begin{enumerate}
    \item \textbf{Formular} el problema correspondiente a la primera fase (método de las dos fases).
    \item \textbf{Resolver} el problema $P$ mediante el método de la matriz completa aplicando el método de las dos fases.
\end{enumerate}
