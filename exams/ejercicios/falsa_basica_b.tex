Dado el siguiente problema de programación lineal:

\begin{equation}
\begin{split}
\mbox{max. } z = 2x_1 + 4x_2\\
s.a.:\\
2x_1 + 4x_2 \leq  80\\
 x_1 +  x_2 \leq 20\\
 x_1 + 3x_2 \leq 100\\
x_1,\,\,x_2 \geq 0\\
\end{split}
\end{equation}

Para una solución en la que $x_1 = 0$ y $x_2 = 10$, se pide calcular la inversa de la base correspondiente a la solución dada.
