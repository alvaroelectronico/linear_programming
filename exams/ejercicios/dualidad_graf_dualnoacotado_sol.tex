Dado el siguiente modelo de Programación Lineal:
\begin{equation}
\begin{split}
\mbox{max. } z = 2x_1 + 1x_2 -5 x_3\\
s.a.:\\
-x_1 -x_2 + x_3 \leq -4\\
x_1 +  x_2 +  x_3 \leq 2\\
x_1,\,x_2,\,x_3 \geq 0
\end{split}
\end{equation}

Obtén su solución óptima resolviendo gráficamente su problema dual.

---

El problema dual es:
\begin{equation}
\begin{split}
\mbox{max. } z = -4y_1+2y_2\\
s.a.:\\
-y_1 + y_2 \geq 2\\
-y_1 + y_2 \geq 1\\
y_1+y_2 \geq -5\\
y_1,\, y_2\geq 0
\end{split}
\end{equation}
