Dado el siguiente problema de programación lineal:

\begin{equation}
\begin{split}
\mbox{max. } z = 2x_1 + 4x_2\\
s.a.:\\
2x_1 + 4x_2 \leq  100\\
1x_1 + 1x_2 \leq 40\\
1x_1 + 3x_2 \leq 80\\
x_1,\,\,x_2\geq 0\\
\end{split}
\end{equation}

Para una solución en la que $x_1 = 10$ y $x_2 = 20$, se pide calcular la inversa de la base correspondiente a la solución dada.
